About This Project

InsightMap is a web-based geovisualisation platform for urban designers, architects, planners, and local councils. It provides interactive tools for exploring land use, heritage, environmental, and planning datasets in Canterbury and Kent. The platform supports evidence-based decision-making, site analysis, and public engagement by integrating authoritative spatial data with a clear, user-friendly interface.

  • Purpose: To enable multi-layered spatial analysis for urban design, helping users understand constraints, opportunities, and context for development and placemaking.
  • Features: Toggleable map layers, feature info sidebar, search, export tools, legends, and measurement utilities.
  • Audience: Urban professionals, students, and the public interested in local planning and environment.

Tutorial

  1. Explore the Map: Use the layer toggles to show/hide features such as heritage, environment, planning, and infrastructure.
  2. Click Features: Click on map features (e.g., buildings, trees, nature reserves) to view detailed info in the sidebar.
  3. Search: Use the search bar to find locations or features by name.
  4. Export: Use the export button to download the current map view as PNG, GeoJSON, or DXF for further analysis.
  5. Legend: Refer to the legend for colour codes and symbols for each layer.

For advanced use, see the project repository for code and documentation.

Design Choices & Features

  • Legible, minimal interface with consistent colour coding for land use and designations.
  • Sidebar displays feature info on click, including heritage, environment, planning, and infrastructure details.
  • Layer toggles, search, and measurement tools encourage exploration without clutter.
  • All data is processed for spatial accuracy and clipped to Canterbury for local relevance.
  • Accessible design: responsive layout, keyboard navigation, and ARIA roles for sidebar.
  • Export and charting tools for further analysis.
