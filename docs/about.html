About This Project

InsightMap is a web-based geovisualisation platform designed for urban designers, architects, and local councils. It provides clear, interactive tools for exploring land use, heritage features, and other spatial datasets, supporting planning and decision-making.

Design Choices

The map interface prioritises legibility, minimal clutter, and consistent colour coding for land use categories. Layer toggles, search, and measurement tools are provided to encourage exploration without overwhelming users.

Disclaimer

The information displayed on this map is for reference purposes only. While efforts have been made to ensure accuracy, the data should not be used as the sole basis for legal or planning decisions.
